# STARBUGS!
---
一个用phaser写的目前比较简陋的H5像素风伪经典小游戏！

## 运行
请安装http-server：

`npm install -g http-server`

并用`http-server`打开文件夹来运行该游戏

### 目前：
+ 马里奥小游戏
+ 俄罗斯方块
+ 弹球
